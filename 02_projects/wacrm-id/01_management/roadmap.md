# Roadmap — WaCRM

## MVP (Now → Month 3)
- Contact CRUD with tags, notes, status
- Follow-up reminder system
- WhatsApp deep link from contact card
- Manual interaction logging
- Xendit subscription (Free / Solo / Team / Business)

## v2 (Month 3–6)
- WhatsApp Business API integration (two-way sync, automated reminders)
- E-commerce integration (Tokopedia/Shopee order sync)
- Mobile app (web-first PMF first)
- Advanced pipeline analytics

## v3 (Month 6–12)
- Multi-channel (Instagram DM)
- Bulk WhatsApp broadcast (compliant via API)
- Integrations with Indonesian payment tools (invoice-app)
