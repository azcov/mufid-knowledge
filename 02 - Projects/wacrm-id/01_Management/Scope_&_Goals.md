# Scope & Goals — WaCRM

## Goal
WhatsApp CRM for Indonesian SMEs. Tag customers, track status, set follow-up reminders — without changing how owners use WhatsApp. Web-first, mobile-first UX.

## In Scope (MVP)
- Contact management: name, phone, tags, notes, last contact date
- Conversation status tracking (custom stages)
- Follow-up reminders (set date/time → owner gets notified)
- Customer history log (manual notes per contact)
- Team sharing (Team + Business tiers)
- Auth: email + Google OAuth
- Xendit subscription billing
- **No WhatsApp Business API** — manual logging only at MVP

## Out of Scope (MVP)
- WhatsApp API integration (Phase 2)
- Automated WhatsApp message sending / blasting
- E-commerce integration (Tokopedia, Shopee)
- Built-in chat interface (never — WaCRM is a layer)
- Mobile app — web-first
- Multi-channel (Instagram DM, email, Telegram) — WhatsApp only
