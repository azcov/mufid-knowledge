## Indonesian Legal Entity Options

### 1. CV (Commanditaire Vennootschap)
| | |
|-|-|
| Cost to register | ~Rp 2–5 juta (notaris) |
| Min. founders | 2 (active + passive partner) |
| Tax | Personal income tax (PPh 21/29) |
| Best for | Small business, freelance studio, early stage |
| Limitation | Cannot raise institutional investment |

### 2. PT (Perseroan Terbatas) — Recommended for long-term
| | |
|-|-|
| Cost to register | ~Rp 5–15 juta (notaris + OSS) |
| Min. founders | 2 shareholders |
| Min. capital | Rp 50 juta (PT Perorangan: Rp 0) |
| Tax | Corporate income tax 22% (PPh Badan) |
| Best for | Serious product company, future investment, credibility |
| Upside | Can raise from investors, issue shares, hire formally |

### 3. PT Perorangan (Single-person PT) ← Easiest to start
| | |
|-|-|
| Cost | ~Rp 0–500K (OSS/online, no notaris required) |
| Min. founders | 1 (you alone) |
| Requirement | Indonesian citizen, UMKM scale |
| Tax | Corporate income tax (0.5% final for revenue < Rp 4.8M/year) |
| Best for | Solo founder going from side project → legit business |
| Limitation | Cannot scale to Tbk (public), limited credibility vs full PT |

## Recommended Progression

```
Now                  Phase 1             Phase 2             Phase 3
────────────         ──────────────      ──────────────      ──────────────
Personal brand   →   PT Perorangan   →   Full PT         →   PT + team
No revenue           First revenue        MRR > Rp 50M        MRR > Rp 500M
Side project         Still solo           Maybe 1-2 hires      Real company
```

**Trigger to register PT Perorangan:** First paying customer, or when you need a business account / NPWP for invoicing.

**Trigger to upgrade to full PT:** When you need to bring in a co-founder, raise money, or hire > 3 people formally.
