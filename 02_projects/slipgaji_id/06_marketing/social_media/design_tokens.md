# Design Tokens — SlipGaji

## Brand Personality
**Professional · Reliable · Simple · Indonesian HR**
Feels like a competent accountant friend — serious but approachable.

## Colors
| Token | Hex | Usage |
|-------|-----|-------|
| Primary | `#1E3A5F` | Deep navy — professional, trust |
| Accent | `#3B82F6` | Blue — tech, clarity |
| Success | `#10B981` | Green — correct calculation ✓ |
| Surface | `#F8FAFC` | Light gray-white |
| Text | `#1E293B` | Near-black |

## Typography
| Role | Font | Weight |
|------|------|--------|
| Headline | Inter | 700–800 |
| Body | Inter | 400 |
| Numbers | JetBrains Mono or Inter | 600 (for salary figures) |

## Visual Style
- **Style:** Clean, corporate-lite. Grid-based, structured like a payslip.
- **Graphics:** Blue + white + subtle grid lines. PDF slip previews are key visuals.
- **Icons:** Document, calculator, BPJS logo context, currency
- **Avoid:** Fun/playful — this is money, keep it serious

## Brand Voice
- Authoritative on regulation (PPh 21, BPJS)
- Simple language — translate complex rules into plain Bahasa Indonesia
- No jargon without explanation
- Positioning: "SlipGaji tahu aturan payroll Indonesia, kamu tidak perlu"
