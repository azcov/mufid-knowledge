# Design Tokens — IndoDev-ID

## Brand Personality
**Tech-native · Transparent · Developer-first · No Corporate BS**
Feels like a tool built by developers, for developers.

## Colors
| Token | Hex | Usage |
|-------|-----|-------|
| Primary | `#0F172A` | Near-black — terminal, developer |
| Accent | `#6366F1` | Indigo — tech, modern |
| Green | `#10B981` | Salary numbers, success |
| Surface | `#1E293B` | Dark card background |
| Light | `#F1F5F9` | Light mode text |
| Muted | `#64748B` | Secondary info |

## Typography
| Role | Font | Weight |
|------|------|--------|
| Headline | JetBrains Mono or Fira Code | 700 (terminal feel) |
| Body | Inter | 400–500 |
| Code / stack | JetBrains Mono | 400 |
| Salary numbers | Inter | 700 (prominent) |

## Visual Style
- **Style:** Dark mode by default — developers prefer dark
- **Data-heavy:** Charts, salary ranges, stack badges (Go, React, Python pill badges)
- **Terminal aesthetic:** Subtle, not overdone — monospace font for key info
- **Avoid:** Stock photos of office workers. No suits. Show code, tools, terminals.

## Brand Voice
- Straight-talking: "Go Senior Backend — Rp 25–40M — remote"
- Developer-native: uses correct tech terms, not dumbed down
- Transparent: salary ranges always shown, no "competitive salary" allowed
- Respectful of developer time — no fluff, clear CTA
