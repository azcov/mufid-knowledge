# Scope & Goals — MenuQR

## Goal
Digital menu platform for Indonesian warung/kafe. Scan QR → see menu with photos → order via WhatsApp with one tap. No app, no hardware, no technical setup.

## In Scope (MVP)
- Menu creation: categories, items, photos, prices
- QR code generation (print-ready PDF)
- Customer-facing menu page (mobile web, works on cheap Android + slow data)
- WhatsApp order routing: pre-filled message with customer order
- Sold-out toggle (live update, no reprint)
- Custom logo + branding (paid)
- Xendit subscription

## Out of Scope (MVP)
- In-app payment (orders go to WhatsApp, payment handled offline: cash/QRIS at counter)
- Kitchen display / POS integration
- GoFood/ShopeeFood integration
- Table reservation/booking
- Loyalty/stamp system
- English menu — Bahasa Indonesia first
