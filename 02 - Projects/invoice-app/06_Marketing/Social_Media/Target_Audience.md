# Target Audience — Invoice App

## Primary — Indonesian Freelancers
| Attribute | Detail |
|-----------|--------|
| Age | 22–35 |
| Role | Desainer, developer, fotografer, content creator, konsultan |
| Pain | Tagih klien via WA, transfer manual, sering telat dibayar |
| Current tool | Excel/Google Sheets, template Word, atau tidak ada |
| Trigger | Klien telat bayar lagi, atau mau terlihat lebih profesional |
| Behavior | Instagram, TikTok, Twitter. Gabung komunitas freelancer Indonesia. |

## Secondary — Small Agencies (2–10 orang)
| Attribute | Detail |
|-----------|--------|
| Pain | Invoice masih manual per klien, tidak ada visibility what's owed |
| Goal | Centralize billing, look professional, reduce chasing |

## Psychographics
- Merasa awkward kalau harus nagih via WA tapi tidak ada cara lain
- Ingin terlihat profesional tapi tidak mau bayar software mahal
- "Gratis kalau tidak ada yang bayar" model sangat menarik — zero risk to try
- Sangat sensitif terhadap payment method: QRIS, GoPay, OVO = must have
- Influenced by: freelancer community content, "cara dapat klien" creators
