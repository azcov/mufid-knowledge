# Roadmap — MenuQR

## MVP (Now → Month 3)
Menu builder, QR generation, customer page, WhatsApp order routing, Xendit billing

## v2 (Month 3–6)
Order history, item analytics (popularity), custom domain, food court operator tier

## v3 (Month 6–12)
Built-in ordering (no WhatsApp redirect), basic table reservation, multi-outlet dashboard

## Future
- GoFood/ShopeeFood integration
- Kitchen display system
- Loyalty / stamp card
