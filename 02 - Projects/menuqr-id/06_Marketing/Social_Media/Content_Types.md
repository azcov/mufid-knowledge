# Content Types — MenuQR

## Pillar 1 — Before/After Demo (50%)
> "Show the transformation in 60 seconds"

| Format | Example |
|--------|---------|
| Reels | "Warung kamu bisa kayak gini dalam 15 menit" |
| Before/After | Foto menu laminated lusuh → QR code clean di atas meja |
| Screen recording | Customer scan QR → lihat menu → tap Pesan via WhatsApp |
| Setup demo | "Cara buat menu digital dalam 15 menit — live" |
