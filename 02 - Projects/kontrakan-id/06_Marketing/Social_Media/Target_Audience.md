# Target Audience — Kostify

## Primary — Individual Landlords (1–10 units)
| Attribute | Detail |
|-----------|--------|
| Age | 35–60 |
| Status | Punya kos atau kontrakan sebagai passive income sampingan |
| Pain | Tidak tahu siapa yang sudah bayar, siapa yang nunggak |
| Current tool | WhatsApp group, buku catatan, atau ingatan saja |
| Trigger | Bulan ini ada yang nunggak 2 minggu dan baru sadar |
| Social behavior | Facebook lebih dari Instagram, aktif di grup komunitas landlord |

## Secondary — Kos Owners (10–20 rooms)
| Attribute | Detail |
|-----------|--------|
| Status | Kos sebagai penghasilan utama atau signifikan |
| Pain | Reminder bayar masih manual dan awkward |
| Goal | Otomatisasi reminder, laporan income bulanan untuk laporan pajak |

## Psychographics
- Tidak ingin tampak demanding kepada penyewa (WA reminder terasa awkward)
- Merasa property management "harusnya ada yang handle" tapi belum tahu solusinya
- Price-sensitive: Rp 49K/bulan masuk akal kalau jelas manfaatnya
- Responds to: "kamu pernah lupa cek siapa yang belum bayar?" (relatable pain)
- Koneksi lewat komunitas Facebook/WhatsApp sesama landlord sangat kuat
