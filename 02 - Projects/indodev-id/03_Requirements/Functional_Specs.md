# Functional Specs — IndoDev-ID

## Developer Features (Free)
- Profile: role, stack, seniority, location, remote flag
- Search: filter by role, stack, seniority, location, remote/hybrid/onsite, salary range
- Apply: one-click with saved profile + optional cover note (< 2 min)
- Status tracking: applied / viewed / shortlisted / rejected

## Company Features
- Free: 1 active post, salary range required (no "competitive salary")
- Paid boosts: featured placement, email blast, candidate search
- Applicant management (status stages)

## Salary Transparency Rule
Salary range is a **required field**. Publish is blocked without it. No exceptions.
This is the core trust signal for developers.

## Cold Start Strategy
Manual seeding of 200+ real listings before launch (salary range included on all).
Target: 50% organic by Month 3.
